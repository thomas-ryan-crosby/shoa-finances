# HOA Financial Review Dashboard

A comprehensive web application for analyzing Homeowners Association (HOA) financial data, deployed via GitHub Pages.

## Features

- **Year Overview**: Summary statistics for income, expenses, and net income across 2023-2025
- **Monthly Spending Analysis**: Visual breakdown of spending by month for each year
- **Biggest Expenses**: Top expense categories across all years
- **Vendor Analysis**: Most expensive vendors with transaction details
- **Expense Trends**: Multi-year trend analysis for major expense categories
- **2026 Budget Planning**: Interactive budget planning tool with suggested amounts based on historical data

## Setup

1. Ensure you have Python 3.x installed
2. Install required dependencies:
   ```bash
   pip install pandas openpyxl
   ```

3. Process the Excel files to generate data.json:
   ```bash
   python process_data.py
   ```

4. Open `index.html` in a web browser or deploy to GitHub Pages

## Deployment to GitHub Pages

1. Push all files to a GitHub repository
2. Go to repository Settings > Pages
3. Select the branch and folder (root or /docs)
4. The site will be available at `https://[username].github.io/[repository-name]`

## File Structure

- `index.html` - Main HTML file
- `app.js` - JavaScript for data visualization and interactivity
- `data.json` - Processed financial data (generated by process_data.py)
- `process_data.py` - Python script to parse Excel files
- Excel files - Source financial data (P&L and Detailed reports)

## Data Sources

The application processes:
- P&L reports (2023, 2024, 2025)
- Detailed expense reports (2023, 2024, 2025)

## Technologies Used

- Chart.js for data visualization
- Vanilla JavaScript for data processing
- Modern CSS for responsive design
